# BME 2810 – Biomedical Engineering Data Science Inquiry

This repository contains Jupyter notebooks and coursework developed for BME 2810 (Biomedical Engineering Data Science Inquiry). The focus of this course is applying data science methods to biomedical datasets, including data analysis, visualization, and interpretation.

## Topics Covered
- Data exploration and preprocessing  
- Statistical analysis  
- Data visualization  
- Introduction to biomedical data analysis workflows  

## 🛠️ Tools & Technologies
- Python (Jupyter Notebooks)  
- Anaconda (for environment and package management)  
- Pandas, NumPy, Matplotlib (data analysis and visualization)  
- R (CourseKata platform, earlier coursework)
